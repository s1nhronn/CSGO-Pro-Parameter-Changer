# CSGO Pro Parameter Changer

Графическая утилита для настройки параметров оружия в **CS:GO**.  
Приложение позволяет редактировать характеристики оружия, хранить несколько конфигураций, переключать язык интерфейса и сохранять изменения прямо в файлы игры.

# WARNING: русский язык на данный момент сломан!

## Возможности

- просмотр и редактирование параметров оружия в таблице;
- добавление нового оружия в конфигурацию;
- удаление оружия из конфигурации;
- переименование оружия;
- сброс параметров выбранного оружия к значениям по умолчанию;
- сохранение текущего набора настроек;
- загрузка конфигурации с диска;
- выбор активной конфигурации из списка сохранённых;
- переименование активной конфигурации;
- восстановление файлов игры к исходному состоянию;
- интерфейс на русском и английском языках.

Часть этих действий реализована через меню приложения, а горячие клавиши в коде привязаны к `Ctrl+S`, `Ctrl+A`, `Ctrl+Z` и `Delete`.

## Как это работает

Приложение хранит данные конфигураций в SQLite-базе `database/database.db`, а текущие параметры записывает в игровые файлы CS:GO. Для каждой конфигурации создаётся отдельная таблица, а затем значения экспортируются в `.ini`.

При загрузке и сохранении программа:
1. читает исходный `defaults/default_items_game.txt`;
2. изменяет параметры нужного оружия;
3. при необходимости обновляет локализованные файлы `csgo_english.txt` или `csgo_russian.txt`;
4. сохраняет результаты в папке игры.

## Требования

- Windows;
- Python 3.x;
- PyQt5;
- установленная CS:GO с доступом к папке игры;

## Установка

1. Скопируйте проект в папку рядом с установленной CS:GO.
2. Установите зависимости:

```bash
pip install PyQt5
```

4. Запустите приложение:

```bash
python main.py
```

## Использование

После запуска откроется главное окно с таблицей оружия и параметров.  
Можно:

- выбрать конфигурацию;
- добавить оружие;
- отредактировать параметры в таблице;
- сохранить изменения в игру;
- экспортировать конфигурацию.

## Горячие клавиши

- **Ctrl + S** — применить/сохранить изменения;
- **Ctrl + A** — добавить оружие;
- **Ctrl + Z** — восстановить исходные значения;
- **Delete** — удалить выбранное оружие.
